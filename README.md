# Campus-Connect 
Campus Connect is a student community platform for buying, selling, renting and finding lost items within campus communities.


# Campus Connect Portal
A modern and responsive student community platform built using HTML, CSS, and JavaScript.

## Features
- Lost & Found section
- Buy & Sell marketplace
- Room/Flat rental listings
- Responsive sidebar navigation
- Search functionality
- Activity updates panel
- Interactive UI with hover effects and animations

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Future Improvements
- Backend integration
- Authentication system
- Database support
